# Sistema de Indicadores Hospitalares
### Rede Hospital Casa

Sistema web para gestao, calculo automatico e analise de indicadores hospitalares com 4 camadas analiticas.

## Arquivos principais

- `sistema.html` - Sistema principal (entrada + calculo + analise + briefing)
- `index.html` - Dashboard com graficos
- `entrada.html` - Formulario de entrada v1

## Indicadores

Taxa de Ocupacao, TMP, Mortalidade, Reinternacao 30d, IRAS, Absenteismo

Como funciona o armazenamento

Os dados são salvos no localStorage do navegador durante o trabalho.
Para persistir definitivamente na pasta do projeto, clique em Exportar dados.json e substitua o arquivo em data/dados.json.

Para migrar dados entre máquinas: exporte o JSON em uma máquina e importe na outra usando o botão Importar JSON.
